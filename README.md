# Sobre mis colaboraciones académicas

En mi perfil de GitHub aparecen repositorios que corresponden a proyectos académicos desarrollados en equipo durante la universidad. A continuación detallo mi rol y las tecnologías utilizadas en cada proyecto.

---

## Redibo – Plataforma web de renta de automóviles

Proyecto académico desarrollado bajo metodología SCRUM para una materia universitaria. Consiste en una aplicación web para la gestión de alquiler de vehículos.

### Backend – redibo-back
**Rol y participación**
- Desarrollo y apoyo en la lógica de backend
- Colaboración técnica activa, aunque los commits fueron centralizados por un integrante del equipo según las instrucciones del curso

**Tecnologías utilizadas**
- Express, TypeScript, Prisma ORM, PostgreSQL, PNPM

### Frontend – redibo-front
**Rol y participación**
- Diseño de mockups en Figma
- Implementación de componentes visuales: barra de buscador y slide de recomendaciones
- Apoyo general al equipo de frontend

**Tecnologías utilizadas**
- Figma, Next.js, React, Tailwind CSS, NPM

**Gestión del proyecto**
- Planificación y seguimiento usando JIRA bajo metodología SCRUM

---

## Proga-movil-UMSS – App móvil de reconocimiento de plantas

Aplicación móvil desarrollada como proyecto académico para reconocimiento de plantas.

### Frontend
**Rol y participación**
- Desarrollo del frontend de la aplicación móvil
- Participación activa en la colaboración grupal, con integración de cambios gestionada por otro miembro del equipo

**Tecnologías utilizadas**
- JavaScript, React Native con Expo
- GitHub Projects (Kanban board)

---

## Multi-Agent Shopping – Sistema multiagente

Proyecto académico enfocado en sistemas multiagente para simular un entorno de supermercado.

**Rol y participación**
- Creación y definición de agentes
- Apoyo en la planificación del comportamiento de los agentes
- Participación parcial en el frontend, con commits centralizados por un compañero

**Tecnologías utilizadas**
- Python, JavaScript, Tailwind CSS, CSS

---

## Nota final

Estos proyectos representan experiencia académica colaborativa, trabajo en equipo y uso de herramientas reales de desarrollo. Los proyectos personales y de autoría propia se encuentran identificados por separado en mi perfil, destacando iniciativa y aprendizaje autónomo.
